# 台球世界 (Billiards World)

## 项目简介
这是一个关于台球运动的静态展示网站，共包含10个页面。网站采用了现代、响应式的设计风格，以墨绿色和木纹色为主色调，营造出经典台球俱乐部的氛围。内容涵盖了台球的历史、规则、器材、技巧以及著名球星介绍。

## 目录结构
```
/1368
├── css/
│   └── style.css      # 核心样式文件
├── index.html         # 首页
├── history.html       # 发展历史
├── rules.html         # 比赛规则
├── snooker.html       # 斯诺克专栏
├── equipment.html     # 器材介绍
├── tables.html        # 球台规格
├── players.html       # 著名球星
├── techniques.html    # 击球技巧
├── etiquette.html     # 台球礼仪
└── contact.html       # 联系我们
```

## 技术栈
- **HTML5**: 语义化标签构建页面结构。
- **CSS3**: 
  - Flexbox & Grid 布局实现响应式设计。
  - CSS 变量管理颜色主题。
  - Hover 效果增强交互体验。

## 如何运行
由于是纯静态网站，您可以直接双击 `index.html` 在浏览器中打开，或者使用 VS Code 的 Live Server 插件进行预览。
