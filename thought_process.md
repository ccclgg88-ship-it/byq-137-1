# 思考轨迹 (Thought Process)

## 1. 任务分析
- **需求**: 创建一个关于台球的10页HTML网页。
- **核心要素**: 页面数量(10), 主题(台球), 结构(HTML+CSS分离), 风格(简单)。
- **页面规划**: 首页、历史、规则、斯诺克、器材、球台、球星、技巧、图集、联系。

## 2. 规划与设计
- **视觉风格**:
  - 确定了"经典台球厅"的主题。
  - 配色方案: 背景深灰/黑 (#1a1a1a), 辅色台泥绿 (#0a5c36), 点缀金色/黄铜 (#ffd700), 木纹色 (#8d6e63)。
- **架构设计**:
  - 采用共用 CSS (`style.css`) 保持风格统一。
  - HTML 结构统一: Header (Logo) -> Nav (导航) -> Main (内容) -> Footer (版权)。
  - 导航栏在所有页面保持一致，并通过 `active` 类高亮当前页。

## 3. 执行阶段
- **步骤 1: 基础建设**: 创建目录结构，编写 `style.css` 定义全局变量和布局网格。
- **步骤 2: 模板开发**: 先开发 `index.html` 作为标准模板，确认 Header/Nav/Footer 的样式。
- **步骤 3: 批量生成**:
  - 制作内容页面 (`history`, `rules`, `snooker`)，填充文本内容。
  - 制作展示页面 (`equipment`, `tables`, `players`)，使用 Grid 布局展示卡片。
  - 制作功能页面 (`techniques`, `gallery`, `contact`)，实现表单和图片网格。
- **步骤 4: 资源填充**: 使用 Unsplash 的高质量图片 URL 填充 Hero 区域和卡片图片。

## 4. 验证与优化
- **问题**: 用户反馈图集页面在更换图片源后仍不满意，决定更换主题。
- **决策**: 删除 `gallery.html` (精彩图集)，新增 `etiquette.html` (台球礼仪)。
- **执行**: 
  - 创建新页面 `etiquette.html`，内容聚焦于台球运动的绅士文化和比赛行为准则。
  - 使用 `sed` 批量替换所有 HTML 文件中的导航链接和文本，确保网站结构的一致性。
- **文档**: 更新项目文档以反映这一结构变更。
